# 🎓 Student Management System

A modern and responsive **Student Management System** built using **React.js**, **Vite**, **React Router**, and **Bootstrap 5**. This project is designed for beginners to understand how to build a complete CRUD-based dashboard application while following a clean folder structure and reusable components.

---

## 🚀 Features

- 📊 Dashboard Overview
- 👨‍🎓 Student Management
- ➕ Add New Student
- 📚 Course Management
- ➕ Add New Course
- 💰 Fee Management
- 💳 Collect Fee
- 💵 Pay Fee
- 🧾 Fee Receipt
- ⚙️ Settings Page
- 🌙 Light & Dark Theme
- 📱 Fully Responsive Design
- 🎨 Bootstrap 5 UI
- 🛣️ React Router Navigation

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project folder

```bash
cd student-management-system
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
